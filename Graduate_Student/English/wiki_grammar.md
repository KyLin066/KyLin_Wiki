# 英语语法

## `第一章·简答句`

### `第一节·汉英句子比较`

中文句子可以没有主语，也可以不用动词作谓语，也可以并没有宾语；并且中文句子主谓宾的顺序没有严格的语法规则，可以随意调整。

一个正常的英语句子必须具备主谓结构，并且主语一定是谓语动作的发出者。如果有宾语的话，宾语通常是谓语动作的承受者。

看几个例子：

`中文：重庆江北机场就要到了。`

错误：Chongqing Jiangbei Airport is arriving.

正确：We are arriving at Chongqing Jiangbei Airport.

`中文：北京下雨了。`

错误：Beijing is raining.

正确：It is raining in Beijing.

`中文：我要去理发。`

错误：I will cut my hair.

正确：I will ask a barber to cut my hair. 或 I will have my hair cut.

`中文：午饭吃了吗你？`

错误：Lunch eat you?

正确：Have you had lunch?

`中文：我的英语说得很好。`

错误：My English speaks well.

正确：I speak English well.

`中文：狗没有找到。`

错误：The dog did not find.

正确：I did not find the dog.

<br>

`总结`汉语和英语句子的差异：

| `区别` | `汉语句子` | `英语句子` |
| :---: | :---: | :---: |
| `1` | 可以没有主谓宾 | 必须具备主谓 |
| `2` | 主谓宾的顺序比较随意 | 主谓宾的顺序相对固定 |
| `3` | 可以不遵循严格语法 | 大多遵循严格的语法 |

<br>

### `第二节·英语句子的基本结构`

**英语的五种基本结构：**

`1.主语+谓语`

主语加不及物的谓语动词构成，常用来表示主语的动作。

<br>

`2.主语+谓语+宾语`

与下面的“主语+谓语（系动词）+表语”一起看

<br>

`3.主语+谓语（系动词）+表语`

`主谓宾和主系表的区别：`

（1）主谓宾的谓语一定是实义动词

（2）主系表的谓语一定是系动词

|  | `系动词主要分为以下六类` |
| :---: | :---: |
| `表示状态` | be动词（连接主语和表语，无意义，用于构成句子） |
| `表示感官` | look，sound，taste，smell，feel（提醒：感官动词只能加形容词） |
| `表示变化` | grow，fall，turn，get，become |
| `表示保持` | stand，keep，stay，remain |
| `表示表象` | seem，appear |
| `表示结果` | prove |

>`PS：有些动词既可以作系动词，又可以作实义动词，作实义动词时意思为：`

get：得到

turn：转动，转弯

grow：成长，发展，种植

fall：下降

keep：继续，养活

stay：逗留

<br>

`4.主语+谓语+间接宾语+直接宾语`

能区分主谓双宾和主谓宾宾补即可，如何区分见以下第5点

<br>

`5.主语+谓语+宾语+宾补`

区分主谓双宾和主谓宾宾补的一个非常简单的方式就是在两个宾语中间，或者在宾语和宾补中间加个系动词，如果读起来语义通顺就是主谓宾宾补；反之，则是双宾。
